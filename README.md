Build Requirements
==================

C Libraries:

 - Allegro 5.2.*
 - Chipmunk 7.0.*
 - ENet 1.3.*

Tools:

 - Bash or equivalent.
 - Inkscape (for asset compilation)

Usage
=====

To create a server:

 1. Open an instance
 2. Press F1
 3. Run the command `host <port> <max_players>`

Then, after everyone has joined, run `map <mapname>`

To join a server, simply type the IP and port in the main menu and press ENTER.
